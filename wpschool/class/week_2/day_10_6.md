```python
class Student():
    
    def __init__(self, name, address):
        self.name = name
        self.address = address
        
    def introduce(self):
        print("저는 {address}에 사는 {name}입니다.".format(
            address = self.address,
            name = self.name,
        ))
```


```python
student = Student("황원주", "신사동 고시텔")
student.introduce()
```
